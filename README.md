разработка Valid


Краткое описание:
Библиотека предоставляет примитивы для описания правил валидации полей, подключения к HTML-формам и запуска проверок в браузере.


Команды для установки и тесты:

```bash
npm install
npm test
```

validity.ts - браузерная валидация
form.ts - работа с формой
core/Form.ts - fluent API правлиа валидации
types/index.ts - создание единой точки входа

V импортирует ипользователь интерфесы: для FormConnector() и Validator()
FormConnector(), связывает класс с формой в html и получает все поля в виде arr.
Validator() валидирует одно поле
